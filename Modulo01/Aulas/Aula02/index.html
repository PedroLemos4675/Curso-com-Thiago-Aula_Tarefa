Tags vistas na aula

Tags de texto e formatação

  • Tag "p": Cria um Parágrafo

  • Tag "a" : Cria um texto com Hyperlink

  • Tag "strong" : Deixa o texto em Negrito

  • Tag "em" : Deixa o texto em Itálico

  • Tag "u" : Deixa o texto Sublinhado

  • Tag "br" : Adiciona uma Quebra de Linha no texto

Títulos

  • Tag "h1": Cria um título de nível 1.
  • Tag "h2": Cria um título de nível 2.
  • Tag "h3": Cria um título de nível 3.
  • Tag "h4": Cria um título de nível 4.
  • Tag "h5": Cria um título de nível 5.
  • Tag "h6": Cria um título de nível 6.
  • Tag "h7": Cria um título de nível 7.

Listas

  • Tag "ol": Cria uma lista ordenada.
  • Tag "ul": Cria uma lista não ordenada.
  • Tag "li": Adiciona um item à lista.

Outras

  • Tag "details": Cria um bloco de informações que alterna entre mostrar e esconder informações. Um sumário, rótulo ou título para o bloco deve ser dado através da tag "summary".
  • Tag "summary": Adiona um sumário (ou título, ou rótulo) à tag "details".

Resumo da aula

Este conteúdo foi direcionado para aprender tags relacionadas a textos e formatação de textos.
Nesta aula conhecemos um pouco sobre textos e formatações de textos em uma página web. Para isto conhecemos algumas tags HTML relacionadas a textos.
E conhecemos, também, a ancoragem, um valor uri que pode ser inserido no atributo href, muito utilizado em tags "a".
