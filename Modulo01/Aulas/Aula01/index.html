Como um página web é construída? O que é HTML e como funciona?

Resumo

Uma página web é construída a partir de arquivos de códigos com extenções que o navegador e o computador é capaz de interpretar. Os principais, mais básicos, são: .html (HTML), .css (CSS) e .js (JavaScript).
Podemos comparar uma página web com um corpo humano. HTML são os ossos, CSS é a pele e os detalhes do corpo e o JavaScript são os músculos. Sendo mais direto, HTML é responsável pela estrutura da página, CSS a estilização e JavaScript a interatividade.

HTML é responsável pela estrutura da página, ou seja, com ele vamos indicar quais elementos da página terá a ordem em que devem ser disponibilizados.

Esta página está apenas em HTML.
